# Security Policy

## Supported Versions

QuickSDFTool is an Unreal Engine plugin. Security fixes are prioritized for the
current stable release line.

| Version | Supported |
| --- | --- |
| 1.x | Yes |
| 0.x preview releases | No |

Version 1.0 is release-verified on Unreal Engine 5.7.x. Compatibility reports
for other Unreal Engine versions are welcome, but unsupported engine versions or
local forks may need project-specific investigation.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues,
pull requests, or discussions.

Use GitHub Private Vulnerability Reporting for this repository when available:

https://github.com/yeczrtu/QuickSDFTool/security/advisories/new

If that link is not available, open a minimal public issue asking maintainers to
enable or provide a private reporting channel. Do not include exploit details,
private project assets, credentials, or sensitive crash data in the public issue.

## What to Include

When reporting privately, include:

- Affected QuickSDFTool version, release tag, or commit.
- Unreal Engine version and operating system.
- A concise description of the vulnerability and impact.
- Reproduction steps, proof of concept, or crash details if safe to share
  privately.
- Any known mitigations or affected workflows.

Maintainers will acknowledge valid private reports as soon as practical, triage
the impact, and coordinate fixes or disclosure through GitHub Security Advisories
when appropriate.
